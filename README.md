# QR Link Generator

Static QR code generator for links.

## Deploy to Layero

Use Layero default settings:

- Build command: `npm run build`
- Output directory: `dist`

The build copies these static files into `dist`:

- `index.html`
- `styles.css`
- `app.js`
- `qr.js`

No server, database, or environment variables are required.
