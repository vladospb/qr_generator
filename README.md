# QR Link Generator

Static QR code generator for links.

## Deploy to Layero

Use these Layero settings:

- Project type: `Static (no build)`
- Build command: empty
- Output directory: `.`

The app is plain static HTML and does not need a build step:

- `index.html`
- `styles.css`
- `app.js`
- `qr.js`

No server, database, or environment variables are required.
